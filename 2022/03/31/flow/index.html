flow-remove-types 移除注解

flow 仅用于编码时找出错误，编码完成后需移除注解

  1. 安装 yarn add flow-remove-types -D

  2. 运行 yarn flow-remove-types src -d dist

  • src 表示当前文件所在目录
  • -d 指定输出目录 如 dist
